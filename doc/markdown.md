# markdown
## 表格
使用 | 来分隔不同的单元格，使用 - 来分隔表头和其他行</br>
表格上必须是空行

name | age
:---- | ---
LearnShare | 12
Mike |  32
### 对齐
在表头下方的分隔线标记中加入 :，即可标记下方单元格内容的对齐方式：

- :--- 代表左对齐
- :--: 代表居中对齐
- ---: 代表右对齐

```
| left | center | right |
| :--- | :----: | ----: |
| aaaa | bbbbbb | ccccc |
| a    | b      | c     |
```
